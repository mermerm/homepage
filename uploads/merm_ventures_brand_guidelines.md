## 1. 브랜드 철학

### 브랜드 에센스

> **"설계된 신뢰 (Designed Trust)"**

merm벤처스가 만드는 모든 브랜드 경험의 중심에는 **신뢰**가 있습니다. 화려함보다 본질, 유행보다 지속성, 겉모습보다 의미를 우선합니다.

### 브랜드 퍼스널리티

merm벤처스는 다음 5가지 성격을 지닌 브랜드입니다.

| 퍼스널리티 | 설명 |
|-----------|------|
| **전문적 (Professional)** | 근거 있는 전략과 검증된 프로세스로 일합니다 |
| **진정성 있는 (Authentic)** | 과장하지 않고 실질적인 가치를 전달합니다 |
| **따뜻한 (Warm)** | 규모와 관계없이 모든 고객을 동등하게 대합니다 |
| **명확한 (Clear)** | 복잡한 것을 단순하게, 단순한 것을 깊게 표현합니다 |
| **성장 지향적 (Growth-driven)** | 고객의 성공이 곧 우리의 성공입니다 |

### 브랜드 포지셔닝

```
         [고급·럭셔리]
               │
               │
[감성·창의적] ──┼────── [전략·전문적]  ◀ merm벤처스
               │
               │
         [저가·대중적]
```

merm벤처스는 **전략적 전문성**과 **접근 가능한 파트너십** 사이의 최적점에 위치합니다. 대기업 수준의 전문성을 중소기업·소상공인이 체감할 수 있는 방식으로 제공합니다.

## 2. 로고 시스템

### 2-1. 메인 로고

merm벤처스의 로고는 **워드마크(wordmark)** 형태를 기본으로 합니다.

```
┌─────────────────────────────────┐
│                                 │
│   merm  VENTURES                │
│   ────────────────              │
│   브랜드를 설계합니다            │
│                                 │
└─────────────────────────────────┘
```

- **merm**: 소문자, 브랜드 고유 서체 적용, Primary Navy 색상
- **VENTURES**: 대문자, 경량 자간 확장, Medium Gray 색상
- **슬로건**: 선택적 사용, 국문·영문 병행 가능

### 2-2. 로고 변형 시스템

| 버전 | 사용 상황 | 비율 |
|------|-----------|------|
| **풀 로고 (가로형)** | 기본 적용, 명함·웹헤더·문서 | 4:1 |
| **풀 로고 (세로형)** | 정방형 공간, 슬라이드 커버 | 2:3 |
| **심볼 단독** | 파비콘, 앱 아이콘, 소형 인쇄물 | 1:1 |
| **워드마크 단독** | 좁고 긴 공간, 펜·우산 등 굿즈 | 8:1 |

### 2-3. 최소 사용 크기

브랜드 가독성 보호를 위해 다음 기준 이하로 사용하지 않습니다.

| 매체 | 최소 크기 |
|------|-----------|
| 인쇄물 | 가로 25mm 이상 |
| 디지털 화면 | 가로 120px 이상 |
| 굿즈·소형 인쇄 | 가로 15mm 이상 (심볼 단독만 허용) |

### 2-4. 보호 영역 (Clear Space)

로고 주변에는 반드시 **로고 높이의 50% (0.5x)** 이상의 여백을 확보합니다.

```
        ↕ 0.5x
   ←0.5x→ [  merm VENTURES  ] ←0.5x→
        ↕ 0.5x
```

어떠한 텍스트, 이미지, 그래픽 요소도 보호 영역 내에 배치할 수 없습니다.

## 3. 컬러 시스템

### 3-1. Primary Colors (주 색상)

브랜드의 핵심을 표현하는 색상입니다. 모든 공식 자료에 일관되게 사용합니다.

#### Primary Navy — 신뢰, 전문성, 깊이

```
█████████████████  HEX  #1B2D4F
                   RGB  27, 45, 79
                   CMYK C85 M72 Y32 K22
                   Pantone 295 C
```

**사용처:** 로고 메인, 주요 헤드라인, 강조 배경, CTA 버튼

---

#### Accent Gold — 가치, 성취, 프리미엄

```
█████████████████  HEX  #C8963E
                   RGB  200, 150, 62
                   CMYK C15 M40 Y85 K5
                   Pantone 7560 C
```

**사용처:** 핵심 포인트 강조, 아이콘 포인트, 고급 인쇄물 금박 대체

---

### 3-2. Secondary Colors (보조 색상)

주 색상을 보완하며 가독성과 레이아웃 구성에 사용합니다.

#### Warm White — 여백, 청결함

```
█████████████████  HEX  #F8F6F1
                   RGB  248, 246, 241
```

**사용처:** 배경, 카드 배경, 텍스트 위 여백

---

#### Medium Gray — 보조 텍스트, 구분선

```
█████████████████  HEX  #6B7280
                   RGB  107, 114, 128
```

**사용처:** 본문 보조 텍스트, 캡션, VENTURES 워드마크

---

#### Light Gray — 섹션 구분, 테이블 배경

```
█████████████████  HEX  #E5E7EB
                   RGB  229, 231, 235
```

**사용처:** 배경 구분, 테이블 행 교차색, 구분선

## 4. 타이포그래피

### 4-1. 서체 시스템

merm벤처스는 한국어·영어 모두에서 **전문성과 가독성**을 동시에 확보하는 서체를 사용합니다.

#### 영문 주 서체 — **Pretendard**

| 굵기 | 용도 |
|------|------|
| Bold (700) | 헤드라인, 로고 워드마크 |
| SemiBold (600) | 소제목, 강조 텍스트 |
| Regular (400) | 본문, 설명문 |
| Light (300) | 캡션, 주석, 날짜 |

> Pretendard는 한글·영문 혼용 환경에서 최적의 일관성을 제공하며, 무료로 사용 가능합니다.  
> 다운로드: github.com/orioncactus/pretendard

#### 국문 보조 서체 — **Noto Sans KR**

디지털 환경에서 Pretendard 미설치 시 대체 서체로 사용합니다.

#### 인쇄 강조용 서체 — **Playfair Display** (영문 한정)

보고서 커버, 고급 인쇄물의 대형 헤드라인에 한해 사용합니다.  
본문, 디지털 콘텐츠에는 사용하지 않습니다.

---

### 4-2. 타입 스케일

| 레벨 | 크기 | 굵기 | 용도 |
|------|------|------|------|
| Display | 48–64px / 24–32pt | Bold | 커버 헤드라인 |
| H1 | 36–40px / 18–20pt | Bold | 페이지 주제목 |
| H2 | 28–32px / 14–16pt | SemiBold | 섹션 제목 |

| Body | 16px / 8pt | Regular | 일반 본문 |
| Caption | 13–14px / 6–7pt | Light | 캡션, 주석 |

### 4-3. 행간 & 자간 기준

| 요소 | 행간 (Line Height) | 자간 (Letter Spacing) |
|------|-------------------|----------------------|
| 헤드라인 | 1.2 – 1.3 | -0.02em ~ -0.01em |
| 본문 | 1.6 – 1.8 | 0em |
| 캡션 | 1.4 – 1.5 | +0.01em |

## 5. 브랜드 보이스 & 톤

### 5-1. 브랜드 보이스 정의

merm벤처스는 **"경험 많은 선배 파트너"** 의 목소리로 말합니다.

- 잘난 척하지 않지만, 근거 없이 겸손하지도 않습니다
- 어렵지 않게 설명하되, 단순화로 본질을 흐리지 않습니다
- 고객을 가르치려 하지 않고, 함께 생각합니다

### 5-2. 상황별 톤 조정

| 상황 | 톤 | 예시 |
|------|-----|------|
| 서비스 소개 | 전문적·자신감 | "10년간 200개 브랜드를 함께 설계했습니다." |
| 고객 상담 | 친근·경청 | "어떤 부분이 가장 고민이세요? 함께 생각해 봅시다." |
| 성과 보고 | 명확·데이터 중심 | "전환율이 전월 대비 35% 향상되었습니다." |
| SNS 콘텐츠 | 따뜻·영감 | "좋은 브랜드는 기억되고, 위대한 브랜드는 그리워집니다." |
| 위기·사과 | 진솔·책임감 | "저희 측 실수였습니다. 빠르게 개선하겠습니다." |

### 5-3. 사용 언어 원칙

**권장**
- 짧고 명확한 문장 (한 문장 = 하나의 메시지)
- 능동형 문장 ("저희가 설계합니다" > "설계가 됩니다")
- 고객 중심 언어 ("귀사의 브랜드" > "우리 서비스")
- 구체적 수치와 사례 활용

**지양**
- 과도한 전문 용어 남발 (설명 없는 영어 약어)
- 근거 없는 최상급 표현 ("최고", "업계 유일", "완벽한")
- 수동형 어미의 남용
- 과장된 감탄사 및 이모지 남발

### 5-4. 핵심 메시지 문구

아래 문구는 공식 커뮤니케이션에서 우선 사용합니다.

- **슬로건:** 브랜드를 설계합니다
- **서브 태그라인:** 중소기업·소상공인의 신뢰받는 브랜딩 파트너
- **영문 슬로건:** We Design Trust.
- **미션 문장:** 모든 사업에는 고유한 이야기가 있습니다. merm벤처스는 그 이야기를 브랜드로 완성합니다.

## 6. 이미지 & 사진 스타일

### 6-1. 사진 방향성

merm벤처스의 시각 이미지는 다음 원칙을 따릅니다.

**✅ 권장 스타일**
- 자연광 또는 자연광에 가까운 조명
- 실제 작업 환경·사람이 담긴 진정성 있는 장면
- 절제된 색감 (채도 낮추기, Warm White 톤 배경 선호)
- 여백이 충분한 구도
- 한국의 실제 중소기업·소상공인 현장

**❌ 지양 스타일**
- 과도하게 보정된 스톡 이미지 느낌
- 화려하고 인위적인 색감
- 외국인 모델 중심 이미지 (타겟 고객과 괴리감)
- 어둡거나 무거운 분위기

### 6-2. 그래픽 요소 스타일

| 요소 | 스타일 방향 |
|------|------------|
| 아이콘 | 선형(Line) 스타일, 2px 굵기, 둥근 모서리 |
| 일러스트 | 미니멀 플랫 스타일, 브랜드 컬러 팔레트 적용 |
| 차트·그래프 | Primary Navy + Accent Gold 조합, 격자선 최소화 |
| 패턴 | 로고에서 파생된 기하학적 패턴, 배경 사용 한정 |

### 6-3. 이미지 오버레이 처리

사진 위에 텍스트를 올릴 경우:
- **Dark overlay:** Primary Navy, 불투명도 50–70%
- **Light overlay:** Warm White, 불투명도 60–80%
- 텍스트 색상은 위 3-4 대비 기준을 반드시 충족할 것

## 7. 디자인 레이아웃 원칙

### 7-1. 그리드 시스템

| 매체 | 컬럼 수 | 거터(Gutter) | 마진 |
|------|---------|-------------|------|
| 웹 (1440px) | 12컬럼 | 24px | 80px |
| 태블릿 (768px) | 8컬럼 | 20px | 40px |
| 모바일 (375px) | 4컬럼 | 16px | 20px |
| A4 인쇄 | 12컬럼 | 5mm | 20mm |

### 7-2. 여백 기준 (Spacing Scale)

8px 베이스의 배수 시스템을 사용합니다.

| 토큰 | 값 | 용도 |
|------|-----|------|
| xs | 4px | 아이콘-텍스트 간격 |
| sm | 8px | 관련 요소 간 최소 간격 |
| md | 16px | 요소 내부 패딩 |
| lg | 24px | 카드 간 간격 |
| xl | 40px | 섹션 내 소그룹 |
| 2xl | 64px | 섹션 간 간격 |
| 3xl | 96px | 페이지 주요 구분 |

### 7-3. 레이아웃 원칙

**정렬**
- 텍스트는 기본 좌측 정렬 (가독성 우선)
- 중앙 정렬은 헤드라인·커버에 한정
- 우측 정렬은 숫자·날짜·표 내부에만 사용

**계층 구조**
- 크기, 굵기, 색상 대비로 시각적 위계를 명확히 합니다
- 한 화면(또는 지면)에 강조 요소는 최대 3개로 제한합니다

**여백 활용**
- 여백은 비어 있는 공간이 아니라 브랜드 정체성의 일부입니다
- 정보 밀도보다 가독성과 집중도를 우선합니다
